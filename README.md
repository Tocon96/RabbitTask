# RabbitMQTask

App requires smtpconfiguration.json file to work correctly. The file should be located in the main directory of the project and consist of following fields:

{<br />
	"Host": "",<br />
	"Port": "",<br />
	"Username": "",<br />
	"Password": "",<br />
	"Ssl": "",<br />
	"From": ""<br />
}<br />
