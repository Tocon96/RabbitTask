# RabbitMQTask

&nbsp;&nbsp;"Username": "",<br />
&nbsp;&nbsp;"Password": "",<br />
&nbsp;&nbsp;"Ssl": "",<br />
&nbsp;&nbsp;"From": ""<br />
}<br />
